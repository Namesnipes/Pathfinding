Info

This is a showcase of the A* algorithm for calculating the shortest path between two points (with portals)

The dark green block is the start of the path

The red block is the end of the path

The orange blocks and blue blocks are portals which connect to eachother

The green blocks are shortest path calculated by the algorithm

Instructions

Drag the mouse to create a wall

Double click in two different positions to create a portal

Refresh to reset the canvas
